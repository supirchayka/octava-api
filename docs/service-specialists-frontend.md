# ТЗ для фронта: вывод врачей на странице услуги

## Цель
Добавить на детальную страницу услуги блок «Врачи, оказывающие услугу» с карточками специалистов и их фото.

## Источник данных
**Endpoint:** `GET /services/:slug`

В ответе уже приходит массив `specialists` (прикреплённые врачи). Каждый элемент содержит данные специалиста и список его услуг.

## Поля, которые нужно вывести
### `specialists[]`
* `id`
* `firstName`
* `lastName`
* `specialization`
* `experienceYears`
* `photo`
  * `url` (ссылка на изображение)
  * `width`
  * `height`
  * `mime`
  * `originalName`

### Рекомендации по отображению
* Отображать карточки в виде сетки.
* В карточке показывать:
  * Фото специалиста (если `photo` = `null`, использовать плейсхолдер).
  * ФИО (`firstName` + `lastName`).
  * Специализацию (`specialization`).
  * Стаж (`experienceYears`, текст «N лет опыта»).

## Пример блока
```
Врачи, оказывающие услугу
[Фото] Анна Иванова — Дерматолог-косметолог, 12 лет опыта
[Фото] Светлана Романова — Врач-косметолог, 11 лет опыта
```

## Дополнительно
* Блок отображать только если `specialists.length > 0`.
* Желательно отсортировать по фамилии на фронте, если нужен стабильный порядок.
