# Промпт для генерации страниц аппаратов

Ты — автор интерфейсов для публичного сайта OCTAVA. Сгенерируй детальные страницы аппаратов и общий каталог, используя реальные ответы API. Обязательно соблюдай JSON-структуры и учитывай все ключи, даже если значения равны `null` или массивы пустые.

## Роуты и ответы
- Список аппаратов: `GET /devices` возвращает массив объектов `{ id, slug, brand, model, positioning, heroImage }`, где `heroImage` может быть `null` или картинкой с `id/url/alt/caption/order` (через `Image`).【F:docs/public-routes.md†L208-L219】
- Карточка аппарата: `GET /devices/:slug` возвращает блоки `device`, `seo`, `hero` (бренд, модель, позиционирование, массив `certBadges`, картинки `images`), `galleryImages`, `inlineImages`, `attachments`, `indications`, `contraindications`, `sideEffects`, `documents`, `faq`, `services`. При отсутствии данных возвращай пустые массивы или `null`; неверный slug даёт 404.【F:docs/public-routes.md†L221-L247】
- Форма записи: `POST /forms/device` принимает `{ name, phone, message?, deviceId?, deviceSlug?, pdnConsent?, utm*? }`, валидирует наличие `deviceId` или `deviceSlug` и возвращает `{ ok: true }` с кодом 201. Ошибка 400 при отсутствии обязательных полей или неправильном slug.【F:docs/public-routes.md†L261-L265】

## Инструкции для генерации
- Для каталога отображай сетку карточек из `/devices` с брендом, моделью, позиционированием и миниатюрой HERO, кликабельную на `/devices/{slug}`.
- На детальной странице отрисуй все блоки из ответа `/devices/:slug`: сертификаты, вложения, списки показаний/противопоказаний/побочных эффектов (с учётом поля `rarity`), документы с файлами, галерею/inline изображения и связанные услуги с ценой `priceFrom`.
- Всегда подключай SEO-метаданные из поля `seo` (meta/og/robots) и корректно формируй ссылки на файлы по `url` из `Image`/`FileMeta`.
- Форму записи привязывай к конкретному аппарату (slug или id из ответа) и отправляй на `/forms/device` с булевым `pdnConsent` и UTM-полями при наличии.
- При 404 от `/devices/:slug` показывай страницу-заглушку и предложение вернуться в каталог.
